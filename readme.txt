Enables automatic domain syncing for WP Ultimo on JungleWP Hosting.
 
== Description ==
 
This Plugin enables automatic domain syncing on JungleWP Hosting for WP Ultimo. With WP Ultimo You will be able to setup a network of premium DIY sites, like WordPress.com, in less than 5 minutes.
 
= Docs & Support =

This Plugin enables automatic domain syncing on JungleWP Hosting for WP Ultimo. With WP Ultimo You will be able to setup a network of premium DIY sites, like WordPress.com, in less than 5 minutes. You can find more detailed information about JungleWP Ultimo on [docs](https://docs.junglewp.com/article/233-getting-started-with-domain-syncing-in-wp-ultimo).

= Privacy Notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* write any user personal data to the database;
* send any data to external servers;
* use cookies.

 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `junglewp-ultimo.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
